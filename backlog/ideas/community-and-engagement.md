# Ideas — Community & engagement

Raw brainstorm. Promote any to `../pending/` to commit to building it.

36. **Playtime leaderboard** — weekly Discord post ranking total time online, built from
    `monitoring/events.csv` join/leave pairs.
37. **Streak tracking** — "🔥 N-day login streak" recognition for regulars.
38. **Discord ↔ in-game accounts link** — let users `/link <gamertag>` so stats and pings
    map to the right person.
39. **Vote-to-start** — when the server is down, friends react to a Discord message to
    request a start; enough votes triggers the bot to bring it up.
40. **Event scheduler** — `/event <when> <desc>` posts an announcement + reminds the
    channel before a planned build night / PvP session.
41. **In-game mailbox** — leave a message for an offline player; delivered via `tellraw`
    on their next join.
42. **Screenshot/clip drop** — a `#gallery` channel where the relayed in-game chat detects
    image links and reposts them as embeds.
43. **Achievements wall** — collect advancements over time into a per-player Discord embed
    they can show off.
